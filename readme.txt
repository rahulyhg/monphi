About
The Monphi CMS is a Content Management System that was created to simplify the design and mofication time of website design and deployment.
________________________________________________________________________________
Requirements
PHP5
Apache htaccess and mod rewrite
MySQL
________________________________________________________________________________
Installation
Unpack the files and copy them to a directory on your website. Use a web browser to access the folder you have installed the application into and follow prompts on the screen.
If you recieve any errors on the installation be sure you do not have duplicate records in the database directed for use in the setup.
