Add Module

Modules link to a seperate PHP file and are integrated into the system through the template edit page. After you have added a module you can link it to a page by editing the page in the template edit section of the administration panel.

Module Name - Is a descriptive name for the module. The module name should be unique. Modular content is added to the site by editing the primary page template and adding open bracket, module name, close bracket. E.g. [module name]

Module Path - Points the the PHP file for the module. Modules should be installed into the "/mod/" folder. The path must be added to the text field as "mod/php_file_name.php"

Load in All Pages - If the module should appear on all pages of the website.

Suppress Template Error Message - Allows you to suppress the template errors which appear when you do not have the [module name] in the page. This is useful for pages like the sitemap that you do not want the module loaded into. If you have any issues with the module loading into a page it is recommended that you turn the option back on.
