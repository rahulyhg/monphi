Sitemap Data

Sitemap data will be used with the sitemap.xml module. Allowing a quick generation of a sitemap for search engines. See also the Dates section as it is also used for the sitemaps (modification date). Based on www.sitemaps.org.

Change Frequency - Fills in the "changefreq" xml variable. Possibilities always, hourly, daily, weekly, monthly, yearly, never. Default is set to "never".

Priority - The priority will fill the priority xml variable. Possibilities are 0.0 to 1.0 default of 0.5.

Hide from Sitemap - This option will hide this page from the sitemap. This would be good for pages that are test pages, admin pages, etc.
