Main Content

Template File Name - The template file name refers to the html file relative to the index.php primary folder. If you have your template files stored in the tpl folder be sure to appened the file name with "tpl/". The default with the installation will be "main.tpl.html".

Title - The "HTML Browser Window" refers to the window title bar. This is where you have the controls to minimize, maximize, close and other options. This is not the template [title] rather your [htmltitle] block which, in its original concept, is used in the html <title> tag which is nested in the <head> tag.

Page Title - "like h1, h2, etc" refers to the template title of [title] not the window title bar. If you have trouble verifiying this please check your html template file for [title].

Main Content - The main content will be your primary page content that replaces your templates [content] block section. The intended use of the content is to store the majority of the webpage data. In use with the template file we now have a consistent base for our website. If you are having issues with this information displaying on the website verify your html template file contains "[content]".

Use BB Code instead of HTML - If you choose not to use HTML data you can use BB code for text formatting instead.
HTML code entered here will be replaced with the html special characters < will be converted to &lt; etc.
The editing interace will assist with the available options.
