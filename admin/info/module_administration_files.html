Module Administration Files

The module administration files section will link the index page to the administration files you have specified.

Admin Add - Used with the Add Path this will link the admin index to the file via the menu system.

Add Path - The file name of the modules add administration page. This should only be the file and not the entire path.

Admin Edit - Used with Edit Path this will link the admin index to the file via the menu system.

Edit Path - The file name of the modules edit administration page. This should only be the file and not the entire path.

Admin Delete - Used with the Delete Path this will link the admin index to the file via the menu system.

Delete Path - The file name of the modules delete administration page. This should only be the file and not the entire path.

Admin Alternative Menu - If the administration files for the module are not loaded in the /admin/ folder you can specify the location by checking this box and populating the "Alternative Menu Path" text field.

Alternative Menu Path - The location of the files listed above. E.g. /mod/modulename/
