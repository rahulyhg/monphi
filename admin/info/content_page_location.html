Page Location

Page Location stores information which relates to the website location.

URL Alias
The URL Alias will determine the url from the base directory you copied the monphi files.

When the input is proccessed it is then forced to be alpha, numeric, dashes, underscores and forward slashes.

a-z, A-Z, or 0-9, -, _, /

Set as default page
This check box will determine what page to display if no URL input is found. E.G. entering the base directory of the monphi installation.

Set as default error page
The htaccess file will send any file input that cannot be found to index.php. The index.php will process the URL and query the URL input against the database. If no data can be found regarding that page a default error 404 page will be displayed. If you prefer to have a custom error message tick the "Set as default error page" and that page will load for all error 404's.
