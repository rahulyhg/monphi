Database and Credentials


You will have to manually create the database to store the data in. 
# mysql> create database database_name;

We will be using two sets of credentials the first will be the basic access allowing "select" requests so we can read database information. The second set of credentials will be used for the administrative access and the creation of the database. The administrative credential set will require "create", "insert", "select" and "update" query access. 
# mysql> grant select on database_name.* to 'user'@'localhost' identified by
	'password';
# mysql> grant create, insert, select, update, delete on database_name.* to 
	'user'@'localhost' identified by 'password';

The information provided will be added to the configuration file conf/dbc.conf.php.
