File Permissions


Here we will check the permissions for files that the setup will be editing. The monphi.conf.php which stores configurations for the CMS engine, the dbc.conf.php which stores the database configurations and the robots.txt which will point search engines to the sitemap.xml file.

If you have shell access you can use the chmod command for this. 

# chmod 666 conf/dbc.conf.php
# chmod 666 robots.txt
