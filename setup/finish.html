The setup is done, no need for these files. We also want to make sure they aren't used to modify the database configurations. 
# rm -rf setup/


Create/Edit the template file

The default template file is main.tpl.html. You can edit this file or make a few modifications to your websites html file with your favorite editor!

The important blocks to have in the template file are

[admin] - For the administrative toolbar.
[content] - This is the main content pulled from the "Main Content" section.
[error] - Location where engine errors will be displayed
[htmltitle] - The title that will be displayed in your windows title bar, this should be between your templates <title> tags.
[includeData] - Used for the output generated by the simple include
[title] - title to be displayed like a "header", say above the main content.
